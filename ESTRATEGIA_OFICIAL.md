## 1. Sumário Executivo

### Identidade da Empresa

**Demarchi MEI** é uma empresa individual de tecnologia especializada em **consultoria técnica, sustentação de infraestrutura em nuvem e automação de processos** para pequenas e médias empresas. Fundada e operada por **Antonio Demarchi**, Engenheiro de Software com 8 anos de experiência prática e atuação em corporações como Itaú, Rede Globo e nstech.

### Missão

> *"Ser o parceiro técnico de confiança que transforma a tecnologia de PMEs em vantagem competitiva real — com custo previsível, entrega disciplinada e foco em resultado."*

### Visão

Tornar-se referência regional no Sul Fluminense como consultoria técnica de alto nível acessível a empresas que ainda não têm acesso ao padrão das grandes corporações.

### Valores

| Pilar | Descrição |
|---|---|
| **Pessoas** | Relacionamento de parceria, não de fornecedor |
| **Inovação** | Tecnologia de ponta aplicada à realidade do cliente |
| **Criatividade** | Soluções sob medida, não produtos de prateleira |
| **Desafios** | Cada problema técnico é uma oportunidade de entrega |
| **Responsabilidade** | Comprometimento com resultado, documentação e transparência |

### Proposta Central

A Demarchi MEI opera no modelo **"Parceria Técnica Evolutiva"**: um retainer mensal fixo de **R$ 1.600/mês** que entrega ao parceiro monitoramento contínuo, automação incremental e presença técnica dedicada — equivalente a ter um Engenheiro Sênior no time, sem o custo de uma contratação CLT.

## 2. Análise de Mercado e Diferenciais

### 2.1 Contexto de Mercado

O Brasil conta com **22 milhões de PMEs ativas** (SEBRAE, 2025), das quais menos de 15% possuem equipe de TI interna estruturada. Com a aceleração da digitalização pós-pandemia, essas empresas adotaram sistemas em nuvem, ERPs, agendamentos online e portais digitais — mas **não acompanharam a maturidade técnica necessária** para operar e sustentar esses ambientes.

No Sul Fluminense — especialmente em Vassouras e região — esse gap é ainda mais pronunciado: empresas de saúde, logística, contabilidade e varejo dependem de sistemas digitais críticos sem qualquer suporte técnico especializado ativo.

### 2.2 Oportunidade Identificada

```
Problema do mercado:
PMEs digitalizadas, mas sem capacidade técnica interna
para manter, monitorar e evoluir seus ambientes de TI.

Solução da Demarchi MEI:
Advisory técnico contínuo de nível corporativo
por um custo fixo e previsível.
```

### 2.3 Análise Competitiva

| Concorrente | Limitação | Vantagem Demarchi MEI |
|---|---|---|
| Freelancer pontual | Sem continuidade, sem SLA | Retainer mensal + SLA definido |
| Agência de TI generalista | Cara, impessoal, sem especialização cloud | Especialista sênior, custo acessível |
| DevOps CLT contratado | R$ 8k–15k/mês + encargos | R$ 1.600/mês, sem burocracia |
| Suporte do fornecedor de sistema | Lento, reativo, sem contexto | Proativo, conhece o ambiente do parceiro |
| Nenhum suporte | Risco permanente de downtime | Prevenção contínua com ROI mensurável |

### 2.4 Diferenciais Competitivos

1. **Credencial corporativa verificável** — experiência real em Itaú, Globo e nstech, aplicada a PMEs locais
2. **Especialização em Cloud AWS de alto nível** — stack equivalente ao de grandes bancos, acessível para empresas locais
3. **Presença física regional** — Sul Fluminense; reuniões presenciais possíveis, construção de confiança local
4. **Formação contínua e atualizada** — Pós-Tech FIAP em DevOps & Arquitetura Cloud (2025–2026)
5. **Modelo orientado a ROI** — toda entrega conectada a resultado financeiro mensurável para o parceiro
6. **Visão full-cycle** — do frontend ao provisionamento de infraestrutura; parceiro técnico completo
7. **Documentação e processos** — nível bancário aplicado ao ambiente do parceiro

## 3. Modelo de Negócio Detalhado

### 3.1 Proposta de Valor — O Que Resolvemos?

A Demarchi MEI resolve **três dores críticas e interligadas** das PMEs:

#### Dor 1 — Instabilidade Operacional
> *"O sistema cai e ninguém sabe o que fazer."*

**Solução — SRE Lite:** Stack de observabilidade (Prometheus + Grafana) com alertas proativos, runbooks de incidentes e relatório mensal de saúde do ambiente. O parceiro nunca é o último a saber de um problema.

#### Dor 2 — Desperdício em Nuvem
> *"Pago caro no AWS mas não sei se estou aproveitando bem."*

**Solução — Cloud Cost Guard:** Monitoramento de custos AWS com dashboards e alertas de anomalia. PMEs que gastam R$ 3k–15k/mês em cloud tipicamente desperdiçam 20–35%. O serviço se paga no primeiro mês.

#### Dor 3 — Estagnação Técnica
> *"Nosso deploy é manual, arriscado e lento. Não temos tempo para melhorar."*

**Solução — DevOps Advisory:** Estruturação e evolução de pipelines CI/CD com GitHub Actions, com 1 automação entregue por mês. Cada mês o ambiente fica mais robusto e confiável.

## 5. Planejamento Estratégico de Tecnologia

### 5.1 Stack Técnica e Sua Função no Negócio

A stack técnica da Demarchi MEI não é apenas ferramental — ela é o **produto entregue ao parceiro**. Cada tecnologia tem papel direto na proposta de valor.

#### Camada de Observabilidade (Coração do SRE Lite)

```
Prometheus  → coleta de métricas em tempo real dos ambientes dos parceiros
Grafana     → dashboards visuais que transformam dados em decisão
Alertmanager→ notificações proativas antes que o problema vire crise
Splunk      → análise de logs para ambientes mais complexos
```

**Impacto no negócio:** O parceiro recebe visibilidade que antes só grandes empresas tinham. Cada alerta disparado e tratado é ROI mensurável no relatório mensal.

#### Camada de Cloud (Cloud Cost Guard)

```
AWS Cost Explorer    → análise nativa de gastos por serviço/região
AWS CloudWatch       → monitoramento de recursos e anomalias
Terraform (IaC)      → infraestrutura como código = ambientes reproduzíveis
EC2/ECS/EKS/Lambda   → domínio dos serviços mais usados por PMEs em cloud
```

**Impacto no negócio:** PMEs que gastam R$ 3k–15k/mês em AWS geralmente desperdiçam 20–35%. A Demarchi MEI identifica e elimina esse desperdício, gerando economia que paga o retainer e sobra.

#### Camada de Automação (DevOps Advisory)

```
GitHub Actions  → pipelines CI/CD automatizados (build, test, deploy)
Docker          → containerização = ambientes padronizados e portáteis
Kubernetes/EKS  → orquestração de containers para ambientes que escalam
Nginx/ELB       → balanceamento de carga e proxy reverso
```

**Impacto no negócio:** Cada automação entregue por mês reduz o risco de erro humano e acelera a entrega. O parceiro ganha velocidade sem precisar de um time de DevOps.

#### Camada de Backend / Desenvolvimento Avulso

```
C# / .NET Core (CQRS, SOLID, Clean Code)  → APIs de alta criticidade
Java (Spring)                              → sistemas corporativos
Python (Django, Flask)                     → automações e scripts RPA
RabbitMQ                                   → mensageria e filas
```

**Impacto no negócio:** Quando o parceiro precisa de desenvolvimento pontual, a Demarchi MEI entrega com padrão bancário — sem precisar contratar outro fornecedor.

#### Camada de Segurança

```
JWT / Identity  → autenticação e autorização de APIs
WAF / IAM       → proteção de ambientes AWS
Blue Team       → visão de segurança ofensiva aplicada à defesa
LGPD Compliance → adequação de dados sensíveis (saúde, finanças)
```

**Impacto no negócio:** Crítico para parceiros dos setores de saúde e financeiro. A experiência na Globo (Blue Team) e no Itaú é credencial direta para esse mercado.

### 5.2 Escalabilidade do Modelo de Negócio

O modelo foi desenhado para crescer sem perder qualidade de entrega:

#### Fase 1 — Fundação (Mês 1–6) · 1 parceiro
```
Dedicação: 10–15h/mês
Faturamento: R$ 1.600/mês
Foco: validar o modelo, construir o primeiro caso de sucesso
Stack: ferramentas gratuitas + tiers free dos serviços cloud
```

#### Fase 2 — Expansão (Mês 7–12) · 2–3 parceiros
```
Dedicação: 25–40h/mês (ainda conciliável com CLT)
Faturamento: R$ 3.200–4.800/mês
Foco: padronizar onboarding, criar templates de relatório e runbooks
Stack: automatizar monitoramento multi-tenant com uma stack centralizada
```

#### Fase 3 — Consolidação (Ano 2) · 4–6 parceiros
```
Dedicação: 40–60h/mês (avaliação de transição para PJ dedicado)
Faturamento: R$ 6.400–9.600/mês
Foco: construir produto SaaS de monitoramento para PMEs
Stack: plataforma própria de dashboard multi-cliente
Limite MEI: atenção ao teto de R$ 81k/ano → migração para ME/EPP
```

#### Fase 4 — Produto (Ano 3+)
```
Modelo: SaaS de observabilidade e DevOps para PMEs
Faturamento: Recorrente por assinatura
Stack: plataforma própria built on AWS, .NET Core backend, Angular frontend
```
